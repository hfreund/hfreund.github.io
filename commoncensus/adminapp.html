Administrative Web App Redesign

Common Benefits Administrator is a core piece of software for Common Census and it's been around for a long time. As the company worked out how this application would grow, I led the design exercise to help envision a bridge from present to future.

Administrative Mode

The majority of the application contains administrative functionality. Most of the features that had been added were poorly organized and buried deep inside of menus. This vision addressed the lack of a planned architecture and also createc a better framework for future feature development.

Administrative Mode

Interview Mode

The other side of this application is a one-on-one interview between an enroller and an individual either in person or over the phone. In this case, providing the best user experience meant improving the time each task takes for the enroller, and also reducing insider jargon for the enrollee.
